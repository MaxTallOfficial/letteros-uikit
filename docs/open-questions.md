# Открытые вопросы

Список зон, которые пока не зафиксированы окончательно.

## Актуальные вопросы

### 1. Pricing Comparison Table

Таблица сравнения тарифов (`pricing-v1-table`) зафиксирована в аудите, но не реализована как компонент. Причина: сложный layout (1300px, active column, SVG checkmarks), сильная привязка к данным. При необходимости потребует отдельной реализации.

### 2. Адаптивное поведение компонентов

Breakpoints зафиксированы в токенах, но не все компоненты в UI kit реализуют responsive-поведение. Header скрывает nav при ≤767px (hamburger → MobileMenu), остальные компоненты пока desktop-only в демо.

### 3. Hero-секции

Каждая страница имеет уникальную hero-секцию. В UI kit нет универсального Hero-компонента. При создании новых страниц hero нужно собирать из примитивов (Typography, Button, Container).

### 4. Swiper/Slider

На сайте используется Swiper для карточек (differents, CTA шаблоны). В UI kit не включён, так как это сторонняя библиотека. При необходимости — подключать отдельно.

### 5. btn-loader

Spinner-анимация на кнопках (1.2s infinite) зафиксирована в аудите, но не реализована в Button компоненте. При необходимости — добавить как состояние `loading`.
