# Контекст проекта

## Что это

Letteros UI Kit — автономный проект, содержащий дизайн-систему сайта [letteros.com](https://letteros.com). Собран на основе аудита реального production-сайта через Chrome DevTools MCP (март 2026).

## Зачем

UI kit является **source of truth** для создания новых страниц Letteros. Любая новая страница должна собираться из токенов и компонентов, зафиксированных здесь.

## Для кого

- **Разработчики команды** — чтобы видеть все доступные элементы в одном месте
- **AI-агенты** — чтобы собирать новые страницы строго по системе, без выдумывания новых сущностей

## Связь с Letteros

Оригинальный сайт Letteros работает на WordPress 6.8 с кастомной темой `letteros-new`, единый CSS-файл `styles-v1.css`, архитектура BEM. UI kit извлекает из этой системы все значения и воспроизводит их в виде React-компонентов и TypeScript-токенов.

UI kit **не заменяет** оригинальный сайт. Он фиксирует визуальную систему as-is и служит основой для расширения.

## Стек

- Next.js 16, React 19, TypeScript 5, Tailwind CSS 4
- Статический экспорт (`output: "export"`) для GitHub Pages
- Деплой через GitHub Actions

## Как использовать

1. UI kit page: `/uikit` — все компоненты и токены с живыми примерами
2. Токены: `src/tokens/index.ts` — TypeScript-константы
3. Компоненты: `src/components/ui/*.tsx` — готовые React-компоненты
4. CSS-переменные: `src/app/globals.css` — custom properties с префиксом `--l-*`
